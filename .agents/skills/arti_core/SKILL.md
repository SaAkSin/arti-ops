# arti_core

## When to use this skill
- 칠판에 해당 에이전트와 관련된 작업 티켓이 할당되어 코딩 및 자율 검증을 수행해야 할 때

## How to use it
- **물리적 격리 준수:** 레거시 코드 보호를 위해 당신은 오직 `.agents/worktrees/arti_core/` 내부에서만 코딩을 진행해야 합니다. 절대 메인 루트 파일들을 직접 수정하지 마십시오.
- **칠판 통신:** 작업의 진행 상태에 따라 `.agents/board/` 내의 7단계 칸반 폴더로 티켓을 물리적으로 이동(`mv`)하며 통신하십시오.
- **자율 검증 강제:** 코드 작성 후 반드시 `.agents/scripts/ralph-loop.sh` 쉘 스크립트를 실행하여 기존 환경 기반의 자율 검증을 통과해야 합니다.
- **🚨 지식 탐색 규약 (Second Brain):** 코딩 전 레거시 코드를 파악하기 위해 직접 `grep`이나 전체 코드를 무작위로 읽기보다, **반드시 터미널에서 `graphify query` 또는 `graphify explain` 도구를 최우선 사용하여 프로젝트의 의존성 위키 맵을 먼저 로드**하여 문맥을 안전하게 파악하십시오.
