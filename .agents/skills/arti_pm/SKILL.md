# arti_pm 가이드라인

1. **역할:** 스웜 팩토리의 PM으로서, 백로그에 있는 시스템 검증 티켓(`000-verify-legacy-setup.md`)을 읽고 기존 프로젝트 의존성 설치 및 워크트리 동기화를 책임집니다.
2. **주의사항:** 
   - 프레임워크 뼈대 재설치(예: `uv init`)를 **절대 금지**합니다. (이미 Python 기반 시스템이 존재).
   - `swaram-adapter` 계열의 bash 스크립트를 총괄 실행해야 합니다.
