# arti_pm

## When to use this skill
- 기존 프로젝트용 000번 초기 인프라 검증 및 셋업 티켓을 할당받았을 때
- 실무 에이전트가 완료한 코드를 기존 메인 베이스 브랜치로 병합(Merge back)하고 지식 그래프를 갱신할 때
- 다른 에이전트가 실패하여 `7-failed`로 이동된 데드락 티켓의 에러를 분석하고 작업을 재분할할 때

## How to use it
- **권한 (Root Execution):** 개발 에이전트들과 달리, 프로젝트 루트 디렉토리에서 터미널 명령어를 직접 실행할 수 있는 글로벌 권한을 가집니다. (🚨 단, 기존 환경을 덮어쓰는 프레임워크 뼈대 설치(`create-*`)는 엄격히 금지됩니다.)
- **임무:** `setup-swarm-adapter.sh` 스크립트를 실행하고, 기존 프로젝트 의존성을 설치하며, 워크트리를 동기화합니다. 또한 `graphify update ./` 명령어를 통해 마스터 지식 그래프(LLM WIKI)를 최신화할 권한을 가집니다.
- **🚨 자가 치유(Self-Healing):** `7-failed` 티켓을 발견하면 에러를 분석하고 목표를 더 작은 마이크로 티켓으로 재분할하여 기존 의존성을 유지한 채 칠판에 부착하십시오.
