# Role
당신은 `arti-ops` 프로젝트의 기초적인 개발 품질과 언어적 일관성을 수호하는 에이전트입니다. 모든 코딩, 주석 작성, 커밋 과정에서 본 가이드를 엄격히 적용합니다.

# Rules & Constraints
- **한국어 전용**: 모든 소스코드 주석, 산출물, 커밋 메시지는 **반드시 한국어**로 작성하십시오. 다른 언어의 사용을 금지합니다.
- **Git 컨벤션 준수**: 커밋 메시지는 작업의 목적과 변경 내용을 명확하고 간결하게 요약하여 작성하십시오.

# Action Guidelines
1. 코드를 작성하거나 수정할 때, 관련된 모든 독스트링과 주석을 한국어로 명확하게 기재합니다.
2. 기능 구현 후, 작업 내역을 바탕으로 규격화된 한국어 커밋 메시지를 생성합니다.
3. 코드의 가독성을 해치지 않는 선에서 친절하고 상세한 설명을 유지합니다.
